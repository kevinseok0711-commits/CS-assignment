1. Which of the following is most likely to be part of a phishing attack?
   a) A coordinated effort by a large group of individuals to overwhelm a web server with HTTP requests
   b) A request for a user to provide personal information in an email or an online form
   c) The use of software that intercepts packets sent on a public wireless network
   d) The use of software to guess a user's password based on publicly available information about the user
   Answer: b

58. Which of the following is most likely to be needed in order for the application to send a notification to user indicating that the user must change the user's PIN?
    a) The data that the user last changed the PIN
    b) The data that the user last entered the PIN
    c) The number of times that the user has changed the PIN
    d) The number of people who have the same PIN as the user
    Answer: a

61. Which of the following is most likely to be a privacy risk associated with using the application?
    a) If an unauthorized individual gains access to the user's device, the individual could gain access to the user's passwords by attempting all possible four-digit PINs.
    b) If an unauthorized individual gains access to the user's public encryption key, the individual could decrypt the document containing the user's passwords.
    c) If an unauthorized individual obtains a user's email address, the individual could overwhelm the user's device with HTTP requests.
    d) If an unauthorized individual successfully guesses a user's password to one account, the individual will be able to access the passwords to all of the user's other accounts.
    Answer: a

63. A company has decided to use teams of programmers who will collaborative to develop software. Which of the following are likely benefits of this decision?
    a) Consultation and communication will minimize the need to document the software.
    b) Teams of programmers are able to develop the software with significantly less testing than programmers working on their own would require.
    c) The software will likely reflect the diversity of talents and perspectives of the team members.
    d) The use of a team will help reduce bias in the software being developed.
    Answer: c,d
